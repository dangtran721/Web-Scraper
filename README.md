# Web Scraper

A backend project built with Node.js and TypeScript that extracts and processes website data automatically.

## Tech Stack
- Node.js
- TypeScript
- Prisma ORM
- JWT Authentication
- Zod Validation
- Docker

## Features
- Automated web scraping using Cheerio
- AI-assisted data classification
- Secure authentication with JWT
- Clean architecture (Controller – Service separation)
- Dockerized development environment

## Getting Started

```bash
git clone https://github.com/dangtran721/Web-Scraper.git
cd Web-Scraper
npm install
npm run dev
